# HTML SVG Icons Project

## Description

Ce projet contient une collection d'icônes SVG utilisées dans une application web. Les icônes incluent des représentations graphiques pour la maison, la recherche, les paramètres et l'utilisateur. Le projet est structuré pour faciliter l'intégration des icônes dans une page HTML.

## Structure du projet

```
html-svg-icons
├── assets
│   └── icons
│       ├── home.svg        # Icône représentant une maison
│       ├── search.svg      # Icône représentant une loupe pour la recherche
│       ├── settings.svg     # Icône représentant un engrenage pour les paramètres
│       └── user.svg        # Icône représentant un utilisateur
├── css
│   └── styles.css          # Styles CSS pour le projet
├── js
│   └── script.js           # Code JavaScript pour les fonctionnalités interactives
├── index.html              # Point d'entrée du projet
└── README.md               # Documentation du projet
```

## Utilisation

1. Clonez le dépôt ou téléchargez les fichiers du projet.
2. Ouvrez `index.html` dans un navigateur pour visualiser le projet.
3. Les icônes SVG peuvent être utilisées directement dans le fichier HTML en utilisant des balises `<img>` ou en les intégrant directement dans le code HTML.

## Fonctionnalités

- Intégration facile des icônes SVG.
- Styles personnalisables via le fichier `styles.css`.
- Interactivité ajoutée grâce au fichier `script.js`.
